# Data Availability Statement — FlashAttention-4 White-Box Runtime Model

The datasets generated in this study — the synthetic FlashAttention-4 forward-pass configuration matrix, the emulator-generated calibration and validation runtimes, and the white-box predictor source code — are available from the corresponding authors on request. A public repository deposit with persistent identifiers is planned before final submission; no accession numbers or DOIs are available at this time.

The figure source data (PNG files) are included in the paper bundle. Public datasets and hardware-rate assumptions reused in the analysis were obtained from cited arXiv preprints (Zadouri et al., 2026; Jarmusch & Chandrasekaran, 2025/2026).
