# Repository Strategy

- Planned repository: a public GitHub repository for code and a Zenodo deposit for the real-hardware measurement CSVs and NCU profiles.
- Identifiers: none assigned yet.
- Versioning: code will be tagged to match the submitted bundle; data deposit will include a README describing the measurement protocol and column definitions.
- License: code under MIT; data under CC-BY-4.0 (subject to final author decision).
- Embargo: none planned.
- Citation: a `CITATION.cff` file will be added before submission.
